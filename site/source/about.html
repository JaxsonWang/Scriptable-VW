关于 Joiner 小组件

Joiner 是开源免费的，由大众系粉丝车主兴趣开发，所有责任与一汽奥迪和一汽大众公司无关。

Joiner 不会收集您的个人账户信息，所有账号信息将存在 iCloud 或者 iPhone 上但也请您妥善保管自己的账号。

Joiner 会不定期推出新功能，如果一汽奥迪或者一汽大众官方推出了小组件，Joiner 将会停止更新与支持。

代码开源

关于有志之士加入开源社区，添加自己一份贡献，同样有更好的意见或者问题反馈在 GitHub 提出！
